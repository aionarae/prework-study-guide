Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Headers must be in order (h1,h2,h3,h4,h5,h6).
  • Some elements are needed by users who use screenreaders.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are 3 ways to style a webpage using CSS: inline CSS, internal CSS, and external CSS
  • External CSS is when you create a separeate CSS file and link it to the HTML file

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: will add all the files to staging
  • git commit -m "message goes here": commits all the changes and adds a description of the commit
  • git pull origin main: pulls the main branch to make sure your code is up to date
  • git push origin branch-name: pushes your commit to a branch
